# .nissan-r34

## Arch Linux
- Archcraft OS (Not needed for this rice)

## Setup:
- Colors: pywal (`wal -i nissan-r34.jpeg` should apply colors)
- Bar: Polybar
- Terminal: Alacritty
- Shell: ZSH (Oh-My-Zsh)
- WM: bspwm
- Hotkeys: sxhkd
- Launcher: Rofi
- Font[0]: JetBrains Mono Nerd Font
- Font[1]: Noto Sans CJK JP
- GTK Themer: Oomox
